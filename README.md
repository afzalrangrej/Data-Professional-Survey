Data Professional Survey Analysis – Power BI Dashboard <br><br>
Project Overview: <br>
This project presents an interactive Power BI dashboard that analyzes survey data collected from data professionals around the world. The dashboard explores trends related to salary, programming languages, job roles, geographic distribution, and career entry difficulty in the data industry.
The goal of this project is to transform raw survey data into clear visual insights that help understand career patterns, salary distribution, and professional satisfaction in data-related roles.<br><br>
Dashboard Overview: <br>
The dashboard contains multiple visualizations designed to highlight important insights from the survey data. <br>
1. Country Distribution of Survey Takers: <br>
  Key Insights: <br>
    1. The United States contributes the largest share of survey participants. <br>
    2. India is the second largest contributor. <br>
    3. Other regions include the United Kingdom, Canada, and other countries. <br>
